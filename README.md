# Flask Backend API

A backend web application built with Flask. This project demonstrates how to build REST APIs, handle requests, and implement CRUD operations.

## Features
- REST API endpoints
- CRUD operations (Create, Read, Update, Delete)
- Request handling and routing
- Simple backend architecture

## Tech Stack
- Python
- Flask

## Purpose
This project was developed to learn backend development and understand how server-side applications work.

## How to run
```bash
pip install -r requirements.txt
python app.py
